# Terminal Resume

[\![WebMCP verified](https://toolindex-api-production.up.railway.app/badge?origin=https%3A%2F%2Fleofle.github.io%2Fterminal)](https://www.webmcpregistry.org)

An interactive, browser-based terminal emulator that serves as a resume. Type commands like `ls`, `cat`, `help`, `whoami`, and `neofetch` to explore.

**Live:** [leofle.github.io/terminal](https://leofle.github.io/terminal)
